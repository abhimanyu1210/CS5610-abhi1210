Backend - MongoDB

Database
  • To store and retrieve data for the application MongoDB instance is setup on Openshift Cloud Server. NodeJS interacts back and forth with this database.
  • The database name is 'healthygobblerdb' and has a total of 5 collections inside it. Out of the 5 collections, only one of them is prepopulated with exercise data. All the other 4 collections are built dynamically as users of the application change anything on the front end
Authentication
  • For authentication, PassportJS is setup on the server side. Local strategy of matching username and password is also setup.
  • Passport JS uses Express' session to maintain context of the logged in user.
