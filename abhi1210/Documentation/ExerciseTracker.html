Exercise Tracker

Exercise Tracker is a page accessible only to registered users where they can select the type of exercise, log its length and check the calories that they burned doing that exercise.

Once this information is available, the user can go ahead and log the workout or navigate back to home page.

The page is divided into 2 sections:

  • Look Up Exercise
    • Users can choose their exercise type from the drop-down list which covers most of the exercises we do.
    • Then they need to input the length of their exercise
    • Finally, the date for which the exercise needs to be logged
  • Calories Burned
    • Once above information is given, the user can calculate the calories that he burned
    • This section is initially invisible and is displayed when the user inputs all the necessary fields and hits the 'Calculate Calories' button
    • The user can then log the number of calories that he burned against a particular date.
