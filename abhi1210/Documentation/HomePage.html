Home Page

 

This is the home page of the application. The home page consists of the slogan of this web-application, navigations and way to try out the web application as a guest.

The navigation options on the home page are:

  • Moving to About Screen - information about the web-application
  • Moving to Contact Us Screen - information of the developer
  • Moving to Search A Meal Screen - trying out the application as a guest.
  • Moving to Login Screen - Login/Register screen is the User Entry point for account holders.

Navigation around these screens are animated to give users of the application a smooth transition.
