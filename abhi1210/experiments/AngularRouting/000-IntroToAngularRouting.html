Introduction To AngularJS - Routing

Description:

This is the first time I have been introduced to Routing using Angular JS. Using Angular JS Routing we can bring up multiple view in a single html page. The interesting part here is that there is no need for a page refresh when we switch between views. This experiments shows navigation between 3 HTML views inside this single HTML page..

Home Page

Go To Page 1 Go To Page 2 Go To Page 3

Code Snippet: 

            var app = angular.module('IntroToAngularRouting', ['ngRoute']);
            app.config(['$routeProvider',
            function ($routeProvider) {
            $routeProvider.
            when('/page1', {
            templateUrl: 'extras/page1.html'
            }).
            when('/page2', {
            templateUrl: 'extras/page2.html'
            }).
            when('/page3', {
            templateUrl: 'extras/page3.html'
            }).
            otherwise({
            redirectTo: '/page1'
            });
            }]);
