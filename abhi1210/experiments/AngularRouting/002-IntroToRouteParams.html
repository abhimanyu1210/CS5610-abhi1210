Angular JS - Using $routeParams

Description:

Angular JS Routing has a powerful tool of passing parameters to contained HTML views. By using $routeParams service you can retrieve the current set of parameters. This is a simple example of passing the phone ID to the the HTML view. The phone id is embedded into the route path and can be shown in the internal HTML view.

Experiment:

# Phone ID Phone Name Actions
1 1234 Motorola X - Gen 2 Open Phone Details
2 5412 Iphone 6 Open Phone Details
3 9874 Samsung Galaxy Open Phone Details

Code Snippet:

                        var routingParams = angular.module('RoutingParams', ['ngRoute']);
                        routingParams.config(['$routeProvider',
                          function ($routeProvider) {
                              $routeProvider.
                                when('/:phoneId', {
                                    templateUrl: 'extras/phoneDetails.html',
                                    controller: 'ShowPhoneDetailController'
                                });
                          }]);
                        routingParams.controller('ShowPhoneDetailController', function ($scope, $routeParams) {
                            $scope.phoneId = $routeParams.phoneId;
                        });
