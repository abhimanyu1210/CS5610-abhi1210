Angular JS Animations Using ng-animate

Description:

This experiment mixes the world of CSS3 animations with Angular JS. In the below experiment, as you type in the letters from any of the songs, the songs filter and in animated fashion. By defining the animations in a CSS file and then adding it to the li tag using ng-animate, the songs that do not match the filter criteria leave by moving right.

Experiment:


  • {{song}}

Code Snippet:

HTML

            < ng-controller="myCtrl">
                        <input type="text" ng-model="search">
                        <ul>
                        <li ng-animate="'animate'" ng-repeat="song in songs | filter:search">
                        <h3>{{song}}</h3>
                            </li>
                        </ul>
            </div>

Javascript

            function myCtrl($scope) {
                $scope.songs = ['Uptown Funk','Thinking Out Loud','Love me Like you do','Four Five Seconds','Sugar', 'Animals', 'Blank Space'];
            }

CSS

            p{
                font-weight:bold;
            }
            #codeSnipp{
                border:1px solid;
            }
            body{
                padding:80px;
            }
            li{
                list-style-type:none;
            }
            .animate-enter, .animate-leave {
                -webkit-transition: 500ms ease-in all;
                -moz-transition: 500ms ease-in all;
                position: relative;
                display: block;
            } 
            .animate-enter.animate-enter-active, .animate-leave {
                left: 0;
            }
            .animate-leave.animate-leave-active, .animate-enter {
                left: 500px;
            }
