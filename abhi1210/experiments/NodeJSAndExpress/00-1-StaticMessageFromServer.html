Description

This is my first attempt to connect to serverApp.js where all the data and all functions reside. This experiment simply makes a GET call to the server and receives a response from the server when running.

Code Snippet:

Client App JS 

                                var app = angular.module("StaticMessageApp", []);
                                app.controller("StaticMessageController", function ($scope, $http) {
                                    $scope.serverMessage = null;
                                    $scope.callServer = function () {
                                        $http.get('/callServer')
                                            .success(function (response) {
                                                $scope.serverMessage = response;
                                            });
                                    };
    
                                });

Server App JS 

                        /* 1st Experiment Calling Server*/
                        app.get('/callServer', function (req, res) {
                            res.send("HELLO FROM SERVER");
                        });
