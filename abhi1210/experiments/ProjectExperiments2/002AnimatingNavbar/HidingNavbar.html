Hiding Navigation Bar on Scroll Down

Description:

As you can see we have a fixed top nav-bar on the top of the page. I'll be using something similar in my project. But fixed top navbars are a hindrance while interacting with the User Interface. I wanted to hide the nav-bar when the user scrolls down on the page. This can be controlled through javascript.

Why this is important?

My Project Dashboard page will be having a live activity feed that displays users activities as well as other users activities whom this user has followed. The nav-bar comes in between while reading the posts and decreses size of the screen. So in this experiment, as you scroll down the nav-bar will hide itself. And similarly, as you scroll up the nav-bar shows back up.

Experiment:

In this experiment, I'll be using javascript/jquery to detect a scroll event. When a scroll down event is fired and the offset from top is more than 100, we hide the navbar. Subsequently, when we scroll up, the navbar shows up. This is fairly important for my project since a fixed top nav bar can decrese the area of vision on the the dashboard page of the project.

Script

                    $(function () {
                        $(window).scroll(function () {
                            // set distance user needs to scroll before we fadeIn navbar
                            if ($(document).scrollTop() > 100) {
                                $('#myNavBar').hide('slide', 500);
                            } else {
                                $('#myNavBar').show('slide', 500);
                            }
                        });
                    });
