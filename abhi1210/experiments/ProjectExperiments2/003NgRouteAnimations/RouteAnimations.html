Animating Between Views - ngRoute

Description:

ngRoute provides us a way to include multiple HTMLs in a single HTML wrapper. By defining ng-view on the page and leaving it blank to fill up, all the defined routes take that space and toggling occurs between views. But the only problem is that this toggling between views is very sudden and not appealing. User interfaces today are built beautifully to keep users engaging with their websites. One of the most important ways to make view transitions appealing is that we provide a mechanism of smooth transfer between views

By defining a directive "ngAnimate" and including all the necessary CSS and JS files, we can define animations between views.

 
 
 

Code Snippets:

HTML

                        <div class="view-animate-container">
                            <div class="view-animate" ng-view>
    
                            </div>
                        </div>

CSS

                                body {
                                        background:url(profile.jpg) center fixed;
                                    }
                                    /*----------------------------------------*/
                                    .view-animate-container {
                                        height:100% !important;
                                        overflow:hidden;
                                    }
                                    .view-animate {
                                        margin-top:80px;
                                    }
                                    .view-animate.ng-enter, .view-animate.ng-leave {
                                        -webkit-transition:all ease-in-out 0.5s;
                                        transition:all ease-in-out 0.5s;
                                        position:absolute;
                                        width:100%;
                                    }
                                    .view-animate.ng-enter {
                                        opacity:0.6;
                                        left:100%
                                    }
                                    .view-animate.ng-enter.ng-enter-active {
                                        opacity:1;
                                        left:0%
                                    }
                                    .view-animate.ng-leave.ng-leave-active {
                                        opacity:0;
                                        left:-100%
                                    }

JS

                            var app = angular.module("RouteAnimationApp", ["ngRoute", "ngAnimate"]);
                                app.config(['$routeProvider', function ($routeProvider) {
                                    $routeProvider.
                                        when('/', {
                                            templateUrl: "Page1.html",
                                            controller: "RouteAnimationController"
                                        })
                                        .when('/page2', {
                                            templateUrl: "Page2.html",
                                            controller: "RouteAnimationController"
                                        })
                                }]);
                                app.controller("RouteAnimationController", function ($scope,$location) {
                                    $scope.nextPage = function () {
                                        $location.url('/page2');
                                    }
                                    $scope.prevPage = function () {
                                        $location.url('/');
                                    }
                                });
