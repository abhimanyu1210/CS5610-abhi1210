Vertical And Horizontal Navigation Bars

1) Objective

The objective of this experiment is to understand how navigation bar works.

2) Description

Navigation bar helps the users of a website to jump from one page to another. Through this experiment, the default provided links like statistics, source, searchtree and others need to be displayed on our homepage. As per my rough sketch for the home-page, my navigation-bar will be vertical placed on the left side of the home-page. Navigation bars can also be styled as per one's preference. Generally navigation-bars have hover effects that show which link is selected.

Code Snippet:

                /*Styling for horizontal navigation bars*/
                    .horNavClass{
                        list-style-type:none;
                        margin:0px;
                        padding:0px;
                        overflow:hidden;
                    }
                    .horNavClass li{
                        float:left;
                        background-color:antiquewhite;
                        padding:10px;
                    }
                    .horNavClass a {
                        font-weight: bold;
                        text-align: center;
                        text-transform: uppercase;
                        text-decoration:none;
                    }
                    .horNavClass a:hover{
                        background-color:yellow;
                        padding:10px;
                    }
                    .horNavClass a:visited{
                        color:brown;
                    }
                    /*Styling for vertical navigation bars*/
                    .verNavClass{
                        list-style-type: none;
                        margin: 0;
                        padding: 0;
                        overflow: hidden;
                    }
                    .verNavClass a:link, a:visited {
                        display: block;
                        width: 120px;
                        color: brown;
                        background-color: burlywood;
                        padding: 20px;
                        text-decoration: none;
                        text-transform: uppercase;
                    }
                    .verNavClass a:hover, a:active {
                        background-color: bisque;
                    }
