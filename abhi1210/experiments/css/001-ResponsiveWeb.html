Responsive Web Design

1) Objective

The objective of this experiment is to learn how to control html rendering based on screen or device being used.

2) Experiment

2.1) Responsive Text

Description:

Initially, the maximum width for set for the text below is 300px(lowest the browser can go to is 328px). The text is allowed to float left which tells it to take as much as space it can take. As we decrease the size of the page, the text adapts itself to the changing width and places itself one below the other.

2.2) Responsive Images

Description:

There are 3 images below: a tiger, a tiger cub and a cat. Depending on the size of the html page, the image toggles. This is done using @media and putting adjacent breakpoints on the page. What we achieve by doing this is that a desktop user will see a tiger, a tablet user will see a tiger cub and mobile user will see a cat.

2.3) Responsive Navigation Bar

Description:

This is a navigation bar from previous experiment. If the horizontal navigation is not made responsive or adaptive to changing screen sizes, it looks shabby and messes up the look-and-feel of the web-site. When screen size goes below a certain width, we display a vertical navigation bar using css which has the same links that were displayed in the navigation bar.

Code Snippet:

                /* Making Text Responsive*/
                        #respText p{
                            width:300px;
                            float:left;
                        }
                        /*Making images responsive*/
                        h3{
                            clear:both;
                        }
                        img{
                            width:100%;
                        }
                        /* When the screen is for desktop*/
                        @media screen and (min-width: 750px){
                            #cubId {
                                display:none;
                            }
                            #catId {
                                display:none;
                            }
                        }
                        /* When the screen is for a tablet*/
                        @media screen and (max-width:750px) and (min-width:500px){
                            #tigerId {
                                display:none;
                            }
                            #catId {
                                display:none;
                            }
                        }
                        /* When the screen is for phone*/
                        @media screen and (max-width: 500px){
                            #tigerId {
                                display:none;
                            }
                            #cubId {
                                display:none;
                            }
                        }
                        /* When screen goes below horizontal navigation bar link*/
                        #verNav{
                            width:100px;
                            display:none;
                        }
                        @media screen and (max-width:975px){
                            #horNav{
                                display:none;
                            }
                            #verNav{
                                display:inline;
                            }
                        }
