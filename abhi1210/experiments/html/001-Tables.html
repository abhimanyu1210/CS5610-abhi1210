Documentation

1) Objective

The objective of this experiment was to explore Tables in HTML5.

2) Description

Displaying data in a tabular format is a very common usage for any kind of web-sites. Tables can be formatted using bootstrap or creating your own CSS styles. Data that we get through open-source API during our projects or future assignments needs to be displayed in some kind of a tabular format. This experiment aims on simply creating tables. Future experiments with CSS explore on styling and making them more responsive.
