Showing And Hiding Part Of Page Using Javascript (Used On Homepage)

Description:

This experiment is using Jquery to show and hide part of the screen. In detail, the page is divided into 2 sections, the menu section and the body section. On button click, we open the menu-side bar. This will also be used on the home page to hold all the quick links.

This is the side-bar menu. You can close this menu by clicking on the button again.

  • Home
  • Projects
  • Resume
  • Contact

This is where the main page content goes. The side-bar section is hidden right now. Click on button below to open/close the sidebar

Code Snippet:

                    var main = function () {
                        $('#toggle')
		                    .click(
			                    function () {
			                        if (parseInt(document.getElementById('sideMenu').offsetLeft, 10) < 0) {
			                            $('.sideMenu').animate({ left: '0px' }, 400);
			                            $('body').animate({ left: '285px' }, 400);
			                        }
			                        else {
			                            $('.sideMenu').animate({ left: '-285px' }, 400);
			                            $('body').animate({ left: '0px' }, 400);
			                        }
			                    });
                    };
                    $(document).ready(main);
