2) Autocomplete And Date Picker

These two widgets are very popular in websites. Autocomplete Widget has a fixed number of tags. On user input, string matches with these tags are carried out to give suggestions to users. Date-picker is one of the widely used widgets. Every form or website has a date-picker widget on it.

4) Tabs

Tabs are similar to accordion widgets where each tab will have some information for the user. When the user clicks on a particular tab, information related only to that tab is visible. Moreover, the space used to show the information is shared by multiple tabs hence displaying more info in less space.

5) Tool-Tips

Tool-tips are simple hints that can be attached to any HTML element. When the mouse pointer hovers over an element with tool-tip on it, a small box displaying personal text is displayed.

Even simple flowing text on the page can have tool-tips. For Example, hover over this link
