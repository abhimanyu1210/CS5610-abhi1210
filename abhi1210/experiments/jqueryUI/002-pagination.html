Pagination Of Simple Text Using Jquery

This experiment shows how pagination can be done using JQuery. We have multiple paragraph tags inside the div. In the jquery, we decide how many paragraph text can be shown on a single page. Currently the number is 3. We have functions in Jquery to directly jump on the page number or even navigating using previous and next links.

Javascript

                $(document).ready(function () {
                    var show_per_page = 3;
                    var number_of_items = $('#content').children().size();
                    var number_of_pages = Math.ceil(number_of_items / show_per_page);
                    $('#current_page').val(0);
                    $('#show_per_page').val(show_per_page);
                    var navigation_html = 'Prev';
                    var current_link = 0;
                    while (number_of_pages > current_link) {
                        navigation_html += '' + (current_link + 1) + '';
                        current_link++;
                    }
                    navigation_html += 'Next';
                    $('#page_navigation').html(navigation_html);
                    $('#page_navigation .page_link:first').addClass('active_page');
                    $('#content').children().css('display', 'none');
                    $('#content').children().slice(0, show_per_page).css('display', 'block');
                });
